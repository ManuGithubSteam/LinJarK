What are the objectives for the 2015 revival?

The 2014 version on this project was mainly a merge of three different things:
It was a project for the many challenged people out there to help them work with Linux, but also a tool who was focused on the deskop user. In addition to that it was also aimed at the raspberry pi community.

This approach did work well but it did not allow me to focus on the real potential and the things i want to work most.

For this reason i have decided over the last few days that the Control Modules for various programs will not receive any updates. Instead i will focus on the raspberry pi idea behind the project and also include some desktop elements.

This does not mean that linjark is of no use anymore for the handicapped people out there. It can still be used and I'm happy to include new modules here. But I will not develop them any further.

Here are some ideas for linjark on the pi modules

    1. Diashow module
    2. Remember todo this... module
    3. Traffic jam module
    4. Alarm clock module (with voice setting)
    5. Check for new mail
    6. Alert map module
    7. Science module
    8. Timer module
    9. Appointment module

    Hope you enjoy the ride like i do.
